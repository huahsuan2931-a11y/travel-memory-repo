# 🦖 Travel Memory

個人旅遊管理系統 — 白日夢冒險家的旅行足跡

> you only live once

## 功能

- Dashboard 四分類（進行中 / 待規劃 / 願望清單 / 已完成）
- 旅行記憶：規劃中每日行程、已完成地點/美食 tag
- 總花費追蹤（多幣別 + 台銀匯率 + 預算上限）
- Checklist 管理 + 模板系統
- 文件 & 提醒（機票 / 簽證 / 保險 / eSIM）+ Google Calendar 串接
- 照片（Google Drive 連結匯入）
- Google My Maps 嵌入
- 自動偵測行程結束

## 使用

直接開啟 `index.html` 即可使用（示範模式）。

要儲存資料到雲端，請看 [GAS_SETUP_GUIDE.md](./GAS_SETUP_GUIDE.md)。
